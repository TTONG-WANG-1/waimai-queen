## ⚠️ 使用限制

### 资质要求
1. **企业认证** - 需要营业执照和企业账号
2. **开发者认证** - 需要通过饿了么开发者审核
3. **店铺授权** - 只能获取已授权店铺的数据
4. **审核流程** - 约 5-10 个工作日

### 技术限制
- **调用频率**: 每分钟/每小时有严格限制
- **数据范围**: 只能获取合作/授权店铺数据
- **需要签名**: 所有请求需要 OAuth 2.0 签名
- **HTTPS 要求**: 必须使用 HTTPS 加密传输
