# 美团 API 集成分析报告

## 📊 API 调研结果

### 访问的文档
- 美团技术服务合作中心：https://developer.waimai.meituan.com/
- 门店类 API：/home/doc/market/11
- 商品类 API：/home/doc/market/12
- 评价类 API：/home/doc/market/13

### 核心 API 列表

#### 1. 门店类 API
| 接口 | 描述 | 是否需要授权 |
|------|------|-------------|
| `poi/getids` | 获取应用已绑定门店的三方门店 ID | ✅ 需要 |
| `poi/mget` | 批量获取门店详细信息 | ✅ 需要 |
| `poi/open` | 门店设置为营业状态 | ✅ 需要 |
| `poi/close` | 门店设置为休息状态 | ✅ 需要 |

#### 2. 商品类 API
| 接口 | 描述 | 是否需要授权 |
|------|------|-------------|
| `retail/list` | 查询门店商品列表 | ✅ 需要 |
| `retail/get` | 查询商品详情 | ✅ 需要 |
| `retail/batchget` | 批量查询商品详情 | ✅ 需要 |

#### 3. 评价类 API
| 接口 | 描述 | 是否需要授权 |
|------|------|-------------|
| `comment/query` | 根据门店 id 批量查询评价信息 | ✅ 需要 |
| `comment/score` | 通过门店 ID 获取当前门店评分 | ✅ 需要 |

## ⚠️ 使用限制

### 资质要求
1. **企业账号** - 需要注册公司，个人无法申请
2. **商家授权** - 只能获取已授权门店的数据
3. **审核流程** - 需要美团官方审核批准（约 5-10 个工作日）
4. **保证金** - 可能需要缴纳保证金

### 技术限制
- **调用频率**：每分钟/每小时有严格限制
- **数据范围**：只能获取自己门店的数据
- **禁止爬取**：不允许爬取非授权门店数据

## 💡 替代方案

### 方案 1: 继续人工收集（推荐）
**优点**：
- 无需资质，立即可用
- 数据质量高（经过筛选）
- 来源多样化（小红书、大众点评等）

**执行方式**：
- 继续收集美食博主推荐
- 定期更新店铺数据库
- 添加更多区域覆盖

### 方案 2: 大众点评开放平台
**网址**：https://dianping.merchants.meituan.com/
- 部分公开数据可访问
- 仍需商家授权获取详细数据

### 方案 3: 高德/百度地图 API
**优点**：
- 个人开发者可申请
- 可获取 POI 信息（店铺位置、名称）
- 免费额度充足

**API**：
- 高德地图：https://lbs.amap.com/api/webservice/guide/api/search
- 百度地图：https://lbsyun.baidu.com/index.php?title=webapi/guide/webservice-placeapi

### 方案 4: 用户贡献模式
- 允许用户提交推荐店铺
- 审核后加入数据库
- 社区共建模式
