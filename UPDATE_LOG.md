## 🎯 收录标准

### ✅ 接收的店铺类型
- 真实存在的品牌名（如"木屋烧烤"）
- 有具体店名（如"白夜有米"）
- 小红书/大众点评博主推荐
- 深圳本地知名连锁
- 用户真实推荐（需验证）

### ❌ 不接收的店铺类型
- 地名 + 类目（如"深大后山烤肉"）
- 虚假品牌（如"哥老宫"）
- 地方小吃泛称（如"沙县小吃"）
- 具体菜品名（如"螺蛳粉"）
- 无法验证的店铺

## 📮 提交新店铺

### 提交方式
通过飞书发送以下信息：

```
店铺名：XXX
推荐菜：XXX、XXX
区域：南山区/福田区/宝安区
具体位置：XXX 地铁站/XXX 大厦
价格：XX 元
来源：小红书/大众点评/朋友推荐
链接：（如有）
```

### 审核流程
1. 收到提交 → 验证店铺真实性
2. 查询小红书/大众点评确认
3. 获取精准坐标（高德地图）
4. 添加到数据库
5. 更新网站

### 更新时间
- 工作日：当天更新
- 周末：次日更新
