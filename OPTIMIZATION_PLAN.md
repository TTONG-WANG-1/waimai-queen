### 2️⃣ 优化评分算法 - 本地店优先

**旧算法**:
```
评分 = 店铺评分×60% + 距离×25% + 外卖×15% + 平价加分
```

**新算法**:
```
评分 = 店铺评分×50% + 距离×20% + 外卖×10% + 
      本地店加分 (20 分) + 特色标签 (10 分) + 平价加分
```

**权重调整**:
- 店铺评分：60% → 50%（降低品牌优势）
- 距离：25% → 20%（适当降低）
- 外卖支持：15% → 10%（基础要求）
- **本地店 bonus**: +20 分（核心优化）
- **特色标签**: +10 分（鼓励有特色的店）
- 平价加分：≤30 元 +15 分（更严格）
